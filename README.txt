Automatically creates and updates WordPress posts from MembershipWorks events. Seamlessly sync your events with your WordPress site.

== Description ==

Transform your MembershipWorks events into beautiful WordPress posts automatically. This plugin connects to your MembershipWorks account and creates posts for each event, keeping them in sync with any changes.

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/membershipworks-events-to-posts`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings > MembershipWorks Events
4. Enter your MembershipWorks API Key and Organization ID
5. Configure your preferred update frequency and default post tags
6. Click "Save Settings"

== Frequently Asked Questions ==

= How do I get updates for this plugin? =

Register at [wpplugins.symphony-ts.com](https://wpplugins.symphony-ts.com). Once registered, your WordPress site will automatically receive update notifications.

= How often are events synchronized? =

You can choose between hourly, twice daily, or daily updates. You can also manually sync events by clicking "Update Events Now".

= What happens to posts when events are deleted? =

Posts are automatically moved to draft status when their corresponding events end. The plugin maintains a record in the database for tracking purposes.

= Can I prevent the plugin from updating existing posts? =

Yes! There's a setting called "Update Existing Posts" that lets you control whether the plugin should update or ignore changes to existing posts.

== Description ==

This plugin connects to your MembershipWorks account and automatically creates WordPress posts from your events. It keeps the posts in sync with any changes made to the events in MembershipWorks.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/membershipworks-events-to-posts` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings > MembershipWorks Events
4. Enter your MembershipWorks API Key and Organization ID
5. Configure your preferred update frequency and default post tags
6. Click "Save Settings"

== Required Settings ==

1. MembershipWorks API Key
2. Organization ID

You can find these in your MembershipWorks account under Settings > API Access.

== Frequently Asked Questions ==

= How often are events synchronized? =

You can choose between hourly, twice daily, or daily updates. You can also manually sync events by clicking "Update Events Now".

= What happens to posts when events are deleted? =

Posts remain in WordPress even if events are deleted from MembershipWorks. You'll need to manually remove any unwanted posts.
